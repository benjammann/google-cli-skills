# gdocs-cli (DEPRECATED)

**Deprecated**: Use [`gdrive-cli`](../gdrive-cli/) instead. It includes all gdocs-cli functionality plus Drive, Sheets, Slides, Forms, and file management.

## Migration

| Old (gdocs) | New (gdrive) |
|-------------|-------------|
| `gdocs create` | `gdrive create-doc` |
| `gdocs format` | `gdrive format-doc` |
| `gdocs append` | `gdrive doc-append` |
| `gdocs get` | `gdrive read-doc` |
| `gdocs parse` | `gdrive parse-md` |
